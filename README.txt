How to run:
    1. Naviagate to the directory in the terminal which the files are located "A4".
    2. Use these commands in the terminal in order;
        a. npm init
        b. npm i express 
        c. npm i pug
        d. npm i mongo
        e. npm i mongoose
        f. npm i path
        g. npm i faker
        h. mkdir database (makes the database)
        i. mongod --dbpath=database (starts the database)
        j. npm i connect-mongodb-session
    3. run node database-initializer.js
    4. run node server.js
    5. copy the link displayed in the terminal and paste it in your browser of choice.
